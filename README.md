## ğŸ“± Usage Guide

### Task Management
1. **Add Tasks**: Type in the input field and click "Weave" or press Enter
2. **Set Priority**: Choose High, Medium, or Low priority
3. **Categorize**: Select from Personal, Work, Health, Learning, or create custom categories
4. **Add Tags**: Use comma-separated tags for better organization
5. **Mark Complete**: Click the checkbox to mark tasks as done
6. **Delete Tasks**: Click the delete button to remove tasks
7. **Filter View**: Use All/Pending/Completed filters to focus on specific tasks

### Drawing
1. **Open Canvas**: Click the "Canvas" tab
2. **Select Tools**: Choose from pen, shapes, eraser, or text tools
3. **Choose Colors**: 
   - Use "Border Color" for shape outlines and lines
   - Use "Fill Color" for shape interiors
   - Use color picker or preset colors
4. **Draw**: Click and drag to create shapes or draw freehand
5. **Save Work**: Click save to store your drawing
6. **View Gallery**: All saved drawings appear below the canvas

### Notes
1. **Create Notes**: Click "New Note" in the Notes tab
2. **Use Templates**: Quick start with diary, journal, or meeting templates
3. **Format Text**: Use the formatting tools for rich text
4. **Search**: Use the search bar to find specific notes
5. **Organize**: Color-code notes for better organization
